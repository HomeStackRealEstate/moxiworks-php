find()

find(array  $attributes = array()) : \MoxiworksPlatform\Group|null

Find a specified Group by name on the Moxi Works Platform.

find can be performed including the Moxi Works Group name and the Moxi Works Agent ID in a parameter array \MoxiworksPlatform\Group::find([moxi_works_agent_id: 'abc123', moxi_works_group_name: 'groupName'])
